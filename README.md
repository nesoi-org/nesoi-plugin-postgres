# @nesoi/postgres

[PostgreSQL](https://www.postgresql.org/) plugin for the [nesoi](https://github.com/nesoi-org/nesoi) framwork, built with [Postgres.js](https://github.com/porsager/postgres).

## Install

```
npm i @nesoi/postgres
```
